QuickWeather es realmente rápido. Sólo tienes que abrir la aplicación y ver el tiempo ahora mismo. No hay pantalla de carga ni tiempo de espera. El tiempo te llega sin basura adicional, sólo información limpia y clara.

Puede agregar tantos lugares al cajón lateral como desee para tener su clima un solo toque de distancia. QuickWeather puede incluso determinar su ubicación actual si usted lo permite (la información de ubicación no se recopila).

QuickWeather es siempre gratis (sin IAPs o anuncios) y siempre de código abierto

Esto es lo que obtienes.
<ul><li>Radar en pantalla completa</li><li>Tiempo actual, temperatura, "sensación" de temperatura, precipitaciones, velocidad del viento, humedad, presión, punto de rocío, visibilidad</li><li>Gráfico de temperatura y precipitaciones actuales y previstas</li><li>Notificación del tiempo actual</li><li>Notificaciones de alertas meteorológicas</li><li>Temperatura y precipitaciones por hora (48 horas)</li><li>Temperatura y precipitaciones por 3 horas (5 días)</li><li>Previsión diaria del tiempo y temperaturas máximas y mínimas (7 días)</li><li>Índice UV diario</li></ul> <li><li>Previsión diaria del tiempo y temperaturas máximas y mínimas (7 días)

QuickWeather funciona con <a href="https://openweathermap.org/">OpenWeatherMap.org</a> o <a href="https://open-meteo.com/">Open-Meteo</a>. 
<ul><li>Si utiliza OpenWeatherMap, QuickWeather requiere una Clave API activa con la suscripción OneCall para funcionar. Si no dispone de una, puede registrarse para obtener una clave API gratuita <a href="https://home.openweathermap.org/subscriptions/unauth_subscribe/onecall_30/base">aquí</a>.</li><li>Si utiliza Open-Meteo, QuickWeather no necesita una clave API.</li></ul>

Permisos necesarios
<ul><li><b>Internet</b> - Necesario para conectarse a Internet y obtener información meteorológica</li></ul>
Permisos opcionales
<ul><li><b>Localización</b> - Necesario para recuperar la ubicación actual del usuario, si se solicita</li><li><b>Localización de fondo</b> - Necesario para recuperar la ubicación actual del usuario para las notificaciones y alertas meteorológicas, si se solicita</li><li><b>Recibir arranque completado</b> - Necesario para reprogramar el servicio de notificación cada arranque</li></ul>
